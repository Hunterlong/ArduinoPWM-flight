# Arduino PWM Flight

This project allows your Arduino to act as a radio receiver/transmitter. It will output PWM just like a normal RC receiver. 

### Controls
Throttle Up - W
Throttle Down - S
Roll Up - D
Roll Down - A
Pitch Up - Q
Pitch Down - Z
Yaw Up - E
Yaw Down - C
Flight Mode Change - 1, 2, or 3
Turn Off Everything - L
